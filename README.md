## 문제 분석

초기에는 연속 프레임 정보를 활용하기 위해 CNN-LSTM 구조를 사용하였습니다.

```text
연속 이미지
↓
CNN
↓
LSTM
↓
조향값 예측
```

하지만 Raspberry Pi 환경에서는 추론 지연이 발생했습니다.

실험 결과 약 32ms 수준의 추론 지연이 발생하였고, 실제 주행에서는 이 지연이 차량 반응성을 저하시켰습니다.

이를 통해

> 정확도보다 실시간성이 더 중요한 병목

이라는 사실을 확인할 수 있었습니다.

```text
카메라 이미지

CNN

조향값 예측
```
